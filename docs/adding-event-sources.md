# Adding a new event source

Each event source is an HTTP-triggered Cloud Function that normalizes an incoming webhook into the common `AgentEvent` schema and publishes it to Pub/Sub.

## Step 1 — Write the function

Create `services/ingest/src/sources/YOUR_SOURCE.js`:

```js
import { validateHmac } from '../lib/validate.js'
import { normalize } from '../lib/normalize.js'
import { publish } from '../lib/pubsub.js'

export async function handler(req, res) {
  // 1. Validate the incoming request (HMAC, token, etc.)
  if (!validateHmac(req, process.env.YOUR_SOURCE_SIGNING_SECRET)) {
    return res.status(401).send('Unauthorized')
  }

  // 2. Normalize to AgentEvent
  const event = normalize({
    source: 'your_source',           // lowercase snake_case
    type: 'your_source.event_type',  // e.g. 'slack.message'
    payload: req.body,
    rawHeaders: req.headers,
  })

  // 3. Publish to Pub/Sub
  await publish(event)

  res.status(200).send('ok')
}
```

## Step 2 — Add a fixture for local testing

Create `services/ingest/fixtures/your-source-event.json` with a sample payload from the source's documentation.

## Step 3 — Add the Cloud Function to Terraform

In `infra/terraform/functions.tf`:

```hcl
resource "google_cloudfunctions2_function" "ingest_your_source" {
  name     = "agentwatch-ingest-your-source"
  location = var.region

  build_config {
    runtime     = "nodejs20"
    entry_point = "yourSourceHandler"
    source {
      storage_source {
        bucket = google_storage_bucket.functions_source.name
        object = google_storage_bucket_object.ingest_source.name
      }
    }
  }

  service_config {
    available_memory = "256M"
    service_account_email = google_service_account.ingest_sa.email
    environment_variables = {
      PUBSUB_TOPIC = google_pubsub_topic.events.name
    }
    secret_environment_variables {
      key        = "YOUR_SOURCE_SIGNING_SECRET"
      project_id = var.project_id
      secret     = google_secret_manager_secret.your_source_secret.secret_id
      version    = "latest"
    }
  }
}
```

## Step 4 — Add the agent workflow

In `services/agent/src/workflows/`, add a file for your event type:

```js
// your-source.js
export const systemPrompt = `
You are an agent handling YOUR_SOURCE events.
When you receive a your_source.event_type event, you should:
1. ...
2. ...
Available tools: ...
`

export function matchesEvent(event) {
  return event.source === 'your_source'
}
```

Register it in `services/agent/src/workflows/index.js`.

## Step 5 — Add the secret to Secret Manager

```bash
echo -n "your-signing-secret" | \
  gcloud secrets create your-source-signing-secret --data-file=-
```

Then reference it in `terraform.tfvars`.

## AgentEvent schema

```ts
interface AgentEvent {
  id: string          // UUID, generated by ingest function
  source: string      // 'gmail' | 'slack' | 'calendar' | ...
  type: string        // 'gmail.email_received' | 'slack.message' | ...
  payload: object     // source-specific normalized data
  rawPayload: object  // original webhook body, for debugging
  receivedAt: string  // ISO 8601
}
```
